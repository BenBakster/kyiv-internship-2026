# Інтернатура — Київ та область · розподіл 2026

Статична сторінка з місцями для лікарів-інтернів. Дві вкладки:
- **Місця в розподілі** — перемикач регіону (Київ / область) + зрізу
  («Уся база (МОЗ ~5400)» ↔ «Заблокований пакет розподілу»), розбивка за спеціальностями та закладами.
- **Хірургія: 5 пріоритетів** — усі бази Києва з хірургією (онко-бази виділені) + як розставити 5 пріоритетів.

**Відкрити:** https://benbakster.github.io/kyiv-internship-2026/

- Джерело: [medvacancy.moz.gov.ua](https://medvacancy.moz.gov.ua/). Знімок станом на 4 червня 2026.
- Публічний портал medvacancy показує лише статус «Активна»; цей зріз ширший.
- Статус вакансії = етап процедури, а не «бюджет/контракт». Остаточний перелік — у кабінеті ЕСРР.
- Заяви на розподіл 25.06–01.07, автоматичний розподіл 03.07.

Адаптивна під смартфон, один файл `index.html`.
