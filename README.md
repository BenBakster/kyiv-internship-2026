# Інтернатура — Київ та область · 2026

Статична сторінка з місцями для лікарів-інтернів у медичних закладах
**міста Києва** та **Київської області** — з перемикачем регіонів,
розбивкою за спеціальностями та переліком закладів.

**Відкрити:** https://benbakster.github.io/kyiv-internship-2026/

- Джерело даних: [medvacancy.moz.gov.ua](https://medvacancy.moz.gov.ua/)
- Зріз: статуси «Активна» + «Передано в ЕСРРІ» (єдина база розподілу), рахуються місця.
- Станом на 4 червня 2026.
- м. Київ — 528 місць; Київська область — 217 місць.

Сторінка самодостатня (один файл `index.html`), адаптивна під смартфон.
