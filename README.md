# Інтернатура — Київ та область · розподіл 2026

Статична сторінка з місцями для лікарів-інтернів у **м. Києві** та **Київській області**:
перемикач регіону + перемикач зрізу **«Уся база (МОЗ ~5400)»** ↔ **«Заблокований пакет розподілу»**,
розбивка за спеціальностями та перелік закладів.

**Відкрити:** https://benbakster.github.io/kyiv-internship-2026/

- Джерело: [medvacancy.moz.gov.ua](https://medvacancy.moz.gov.ua/).
- «Уся база (МОЗ ~5400)» = статуси «Активна» + «Передано в ЕСРРІ» (методика МОЗ).
  «Заблокований пакет розподілу» = лише «Передано в ЕСРРІ» — пакет, зафіксований на дедлайні (≈30.05).
- Публічний портал medvacancy показує лише статус «Активна»; цей зріз ширший. Остаточний перелік — у кабінеті ЕСРР.
- ⏳ Знімок станом на 4 червня 2026; заяви на розподіл 25.06–01.07, розподіл 03.07. Статус ≠ «бюджет/контракт».

Адаптивна під смартфон, один файл `index.html`.
