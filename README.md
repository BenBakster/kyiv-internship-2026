# Інтернатура — Київ та область · розподіл 2026

Статична сторінка з місцями для лікарів-інтернів у **м. Києві** та **Київській області**:
перемикач регіону, перемикач зрізу (**уся база розподілу** ↔ **вже в ЕСРРІ**),
розбивка за спеціальностями та перелік закладів.

**Відкрити:** https://benbakster.github.io/kyiv-internship-2026/

- Джерело: [medvacancy.moz.gov.ua](https://medvacancy.moz.gov.ua/).
- ⏳ Перелік ще формується: вакансії передаються в ЕСРРІ, база наповнюється приблизно до 12.06.2026,
  остаточний перелік для подання заяв — до 25.06–01.07.2026. Сторінка — знімок станом на 4 червня 2026.
- Статус вакансії = етап процедури розподілу, а не «бюджет/контракт» (фінансування залежить від категорії випускника).

Адаптивна під смартфон, один самодостатній файл `index.html`.
